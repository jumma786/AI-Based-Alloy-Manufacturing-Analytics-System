# AI-Based Alloy Manufacturing Analytics System

## Project Overview

This project applies Artificial Intelligence and Machine Learning concepts to alloy ingot manufacturing processes.

The system focuses on:

* Predicting alloy purity levels
* Predicting waste material generation
* Improving operational efficiency
* Reducing manufacturing waste
* Supporting Industry 4.0 and smart manufacturing systems

---

# Business Problems

## 1. Alloy Purity Prediction

The system predicts whether alloy ingots will meet required purity standards using industrial manufacturing and sensor-based features.

### Objectives

* Improve product quality
* Reduce rejected batches
* Improve manufacturing consistency

---

## 2. Waste Material Prediction

The system predicts waste material generation during manufacturing operations.

### Objectives

* Reduce production waste
* Improve operational efficiency
* Lower manufacturing costs

---

# Machine Learning Problem Types

| Target Variable   | Problem Type   |
| ----------------- | -------------- |
| purity_pass       | Classification |
| purity_percentage | Regression     |
| waste_material_kg | Regression     |
